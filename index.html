Interactive Recommender CLI

Run the interactive CLI:

python ai_solution_recommender.py

Answers 5 questions: (1) Industry, (2) Problem type, (3) Budget, (4) Technical level, (5) Data sensitivity. Gets ranked recommendations + free alternatives + platform info.

E-commerce Chatbot (low budget): Mistral Large - open source, $30/mo, Le Chat web + API.
Healthcare Documents (high privacy): Claude 3.5 Sonnet - best analysis, Claude web + API.
SaaS Code Generation: Claude 3.5 Sonnet - 92 coding score, Claude web.
Marketing Content: GPT-4o - strong creative + multilingual, ChatGPT web.
